# dgm

PC画面に万華鏡キットを当ててのぞくための、動く幾何学模様を表示する小さなブラウザ作品です。

## 使い方

1. `index.html` をブラウザで開く
2. 左上のパネルで模様を変える
3. 画面に万華鏡キットを当ててのぞく

キーボード操作:

- `f`: 全画面
- `h`: パネルの表示切り替え

## ファイル構成

- `index.html`: 画面の見た目
- `style.css`: 色やレイアウト
- `script.js`: 模様が動くプログラム

## 改造の入口

- `script.js` の `settings.speed` を変えてみる
- `palettes` の色コードを好きな色に変える
- `drawPetals()` をまねして新しい `draw...()` 関数を作る
